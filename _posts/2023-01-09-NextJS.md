## Route

사용자가 접속하는 경로와 우리가 설정하는 경로의 주소차이

http://a.com/ => /pages/index.tsx

http://a.com.sub/ => /pages/sub/index.tsx

http://a.com/sub/about => /pages/sub/about.tsx

http://a.com/sub/1 => /pages/sub/[id].tsx

http://a.com/sub/2 => /pages/sub/[id].tsx
